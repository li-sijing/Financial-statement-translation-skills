## Skill 1: `chinese-financial-translation`

适用于从多份 Word 参考文档中提取术语、对比版本差异、解决术语冲突。

**主要功能：**
- 从 `.docx` 文件中提取段落和表格术语
- 多版本冲突检测与优先级排序（硬规则 > 最新文档 > 早期文档）
- 生成术语对照表，标记未解决的冲突待用户确认

## Skill 2: `financial-statement-translation`

整份财务报表附注（DOCX）及正式报表（XLSX）的中译英翻译工作流。

**主要功能：**
- **附注翻译（DOCX）**：逐段逐格翻译，保留格式（字体、加粗、缩进）
- **报表翻译（XLSX）**：12 张正式报表的自动翻译（含公式保留）
- **格式处理**：Times New Roman / Arial Narrow 字体，行 1-4/1-7 表头加粗，正文加粗逐格匹配中文原版
- **delegate_task 并行翻译**：支持用子代理批量翻译大文件
- **27 个实战陷阱总结**：从实际翻译中积累的易错点与应对策略

## 关键术语规则

| 中文 | 英文 |
|------|------|
| 账面余额 | Book balance |
| 账面价值 | Book value |
| 本期/上期金额 | Current / Previous amount |
| 期初/期末余额 | Opening / Closing balance |
| 短期薪酬 | Short-term employee benefits |
| 坏账准备 | Provision for bad debts |
| 永续债 | Perpetual bonds |
| RMB 规则 | 万元换算用 `RMB X million/billion yuan`；普通金额用 `X yuan` |

完整术语表见 `translation/financial-statement-translation/references/translation-dictionary.md`
